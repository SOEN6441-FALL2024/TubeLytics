# TubeLytics

## Setup

### Adding Git Hooks

To set up the Git hooks, run the following script:

```sh
./git-hooks/add-hooks.sh
```
### Running Unit Tests
```sh
sbt test
```

### Running the Application
```sh
sbt run
```

### Compiling the Application
```sh
sbt compile
```
